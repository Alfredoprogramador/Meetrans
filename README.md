# Meetrans

App web simples que faz tradução em vivo dentro de uma transmissão de vídeo conferência.

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Escolha o idioma de origem e o idioma de destino.
3. Clique em **Iniciar tradução ao vivo** e permita acesso ao microfone.
4. O texto original e a tradução aparecem na sobreposição da transmissão.

## Observações

- O reconhecimento de voz usa a API Web Speech (disponibilidade depende do navegador).
- A tradução usa o serviço público `libretranslate.com`.
